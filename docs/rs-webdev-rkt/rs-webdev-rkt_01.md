在这部分，你将学习关于 Rust 编程语言的知识，包括其基础知识、如何在你的操作系统上安装它，以及如何使用 Rust 工具和包注册表。你还将创建一个 Rocket 网络应用程序，并对其进行配置、编译和运行。我们将探讨为网络应用程序安装和包含其他包的方法。我们将使用 sqlx 来实现网络应用程序与关系型数据库的连接。

+   *第三章**，Rocket 请求和响应*
